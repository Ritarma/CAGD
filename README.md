# CAGD
B-spineDrawing Project for CAGD lesson

这是我写的第一个较大的程序，总写了一千多行，记录2018年自学编程半年的结果吧。

#### 程序功能
使用OpenGL的C#封装库构建的WPF可视化程序，用于B样条曲线，曲面的绘制和基函数图像的显示。
